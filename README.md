# Take no prisoners, comrades.

This GMod addon focues on recreating various ***Call of Duty HUDs*** for use within Sandbox.

## Requirements
There are no requirements. All content is included in this Git.

## Features
**Fully recreated HUDs from various CoD titles**
- Weapon Info
- Score Info
- Scoreboard
- Medals
- Chat
- Challenges

And so much more.

## Regarding localization pull requests
Wanna help translate the addon? Great! Just create a fork, do your translations, and send out a pull request.

- Localizations are stored in `resource/localization` inside `.properties` files. These can be opened in any text editor.
- When contributing to a language that already has a translator (or translators), please be mindful not to change existing strings.
- Work-in-progress translations are to be published as PR drafts. Also applies if you wish for input from other contributors.
- Various elements in the texts are to be kept as-is; `%s` for example.
